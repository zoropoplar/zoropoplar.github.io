# 1. 基本概念

# 1.1 什么是数据结构？

数据结构（Data structure），存储数据的不同方式。

# 1.2 什么是算法？

算法（Algorithm），同一问题的不同解决方法，算法往往是针对特定数据结构的。

# 1.3 算法的复杂度

算法的复杂度分为时间复杂度、空间复杂度。 通常使用 Big O 表示法来描述算法的复杂度。

# 2. 常见的数据结构

# 2.1 线性结构
