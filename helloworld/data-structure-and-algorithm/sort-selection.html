# 选择排序

# 算法介绍

算法思想：每次在找到最小值，放到最前面。

  • 平均时间复杂度：n^2
  • 最坏时间复杂度：n^2
  • 最好时间复杂度：n^2
  • 空间复杂度：1
  • 稳定性：不稳定

# 代码实现

package com.yangzhuo.demo.base.algorithm;

import java.util.Arrays;

/**
 * Description: 选择排序
 *
 * 算法思想：
 * 1. 找到最小值，放到数组第一个位置
 * 2. 找到第二小值，放到数组第二个位置
 * 3. 依此类推
 *
 * @author yangzhuo
 * @date 2024/5/16 07:23
 */
public class Demo01_SelectionSort {
    public static void main(String[] args) {
        int[] arr = {5, 2, 4, 1, 3, 7, 9, 8, 6};
        selectionSort(arr);
        System.out.println(Arrays.toString(arr));
    }

    private static void selectionSort(int[] arr) {
        for (int i = 0; i < arr.length - 1; i++) {
            // 找到最小值
            int minIndex = i;
            for (int j = i + 1; j < arr.length; j++) {
                if (arr[j] < arr[minIndex]) {
                    minIndex = j;
                }
            }
            // 放到数组前面
            if (minIndex != i) {
                swap(arr, i, minIndex);
            }
        }
    }

    private static void swap(int[] arr, int i, int j) {
        int temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }
}

# 优化改进

优化思路：每次遍历找到最小值和最大值，最小值放在最前面，最大值放到最后面。

/**
 * 优化后的选择排序
 */
private static void selectionSort2(int[] arr) {
    for (int i = 0; i < arr.length / 2; i++) {
        // 找到最小值
        int minIndex = i;
        int maxIndex = arr.length - i - 1;
        for (int j = i + 1; j < arr.length - i - 1; j++) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j;
            }
            if (arr[j] > arr[maxIndex]) {
                maxIndex = j;
            }
        }
        // 放到数组前面
        if (minIndex != i) {
            swap(arr, i, minIndex);
        }
        if (maxIndex != arr.length - i - 1) {
            swap(arr, arr.length - i - 1, maxIndex);
        }
    }
}
